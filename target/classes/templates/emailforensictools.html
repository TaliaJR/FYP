Computer forensic software tools can be used to aid a phishing e-mail investigation by interpreting the e-mail header, recovering deleted e-mails and performing e-mail server analysis. Encase, one of the forensic tools that can be used in e-mail forensics, has the ability to analyse the e-mail header to find the IP address of the machine that sent the e-mail and the IP address of the sender's SMTP server. Encase also includes a feature that allows related e-mails to be grouped together (known as e-mail threading) to help the investigator understand the context of the e-mails. FTK, another forensic tool, includes a feature that allows e-mails to be parsed for certain words along side the ability to analyse e-mail headers for IP addresses.
